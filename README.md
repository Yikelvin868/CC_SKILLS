# CC_SKILLS

A comprehensive collection of **168+ Claude Code Skills**, **18 Agents**, and **32 Commands** for AI-powered development, marketing, and productivity workflows.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Yikelvin868/CC_SKILLS.git

# Run setup script
cd CC_SKILLS
chmod +x setup.sh
./setup.sh
```

Or manually copy to your Claude Code configuration:
```bash
cp -r skills/* ~/.claude/skills/
cp -r agents/* ~/.claude/agents/
cp -r commands/* ~/.claude/commands/
cp rules/* ~/.claude/rules/
```

## Configuration

### CLAUDE.md Principles

1. **Agent-First** - Delegate complex tasks to specialized agents
2. **Parallel Execution** - Use Task tool for concurrent work
3. **Planning First** - Plan before execution
4. **TDD Driven** - Test-driven development
5. **Security First** - Security as priority

### Code Standards

- Test coverage >= 80%
- File size: 200-800 lines
- Conventional Commits
- No emojis in code
