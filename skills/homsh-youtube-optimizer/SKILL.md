# HOMSH YouTube Optimizer — 虹识 YouTube 频道优化技能

## Overview

为 HOMSH Technology 的 YouTube 频道优化视频元数据、SEO、缩略图策略，最大化产品视频的搜索曝光和获客转化。

## When to Use This Skill

- 视频制作完成后，需要生成 YouTube 上传所需的标题/描述/标签
- 需要分析竞品频道的 SEO 策略
- 需要规划 YouTube 内容日历

## Workflow

### 1. 标题优化

**公式：** `[产品核心卖点] | [技术关键词] | [品牌名]`

**规则：**
- 70 字符以内
- 前 40 字符包含核心关键词
- 避免全大写（除品牌名/缩写）
- 包含数字或量化指标（"Sub-1s", "3-in-1"）

**模板：**
```
Iris + Face Recognition Turnstile Gate | Sub-1s Biometric Access | HOMSH
3-in-1 Biometric Gate: Iris + Face + Fingerprint | Live Demo | HOMSH
[产品名] — [核心差异化] | [应用场景] | HOMSH Technology
```

### 2. 描述优化

**结构：**
```
第1行：一句话价值主张（包含主关键词）
第2行：空行
第3-5行：产品核心功能（3个要点）
第6行：空行
第7行：应用场景
第8行：空行
第9行：CTA + 联系方式
第10行：网站链接
第11行：空行
第12行+：时间戳章节标记（如适用）
```

**示例：**
```
HOMSH iris recognition turnstile gate delivers triple biometric verification 
(iris + face + fingerprint) in under 1 second.

Key Features:
- Sub-1-second multi-modal biometric verification
- Built-in tamper alarm and anti-tailgating detection

Ideal for: Airports, Government Buildings, Data Centers, Military Facilities

Contact us: sales@opticsiris.com
Website: https://www.opticsiris.com

0:00 — The Problem with Traditional Access Control
0:08 — Biometric Identity Verification
0:14 — Product Introduction
0:20 — Live Demo: 10 Users Walk Through
```

### 4. 缩略图策略

**设计规则：**
- 1280x720 最小分辨率
- 高对比度，深色背景
- 产品特写 + 大字标题
- 避免文字过多（手机端看不清）
- 品牌红色作为点缀色

**推荐帧截取时间点：**
- 设备屏幕显示人脸识别界面（最直观）
- 绿色 LED 闸门打开瞬间（动感）
- 虹膜扫描特写（科技感）

### 5. 发布时间

**B2B 安防行业最佳发布时间：**
- 周二至周四
- UTC 14:00-16:00（覆盖亚洲下午 + 欧洲上午 + 美国早晨）

### 6. 播放列表规划

```
1. Product Demos（产品演示）
2. Technology Explainers（技术讲解）
3. Installation Guides（安装指南）
4. Case Studies（案例分享）

```
