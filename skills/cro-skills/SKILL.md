# CRO (Conversion Rate Optimization) Skills

Comprehensive conversion rate optimization toolkit for marketing pages, signup flows, onboarding, forms, popups, paywalls, churn prevention, and A/B testing.

## When to Use

Use this skill when:
- Analyzing and optimizing marketing pages for conversions
- Improving signup and onboarding flows
- Optimizing forms for higher completion rates
- Designing effective popups and modals
- Improving paywall and upgrade screens
- Reducing churn and improving retention
- Setting up and analyzing A/B tests
- Implementing analytics tracking

## Included Capabilities

| Area | Description |
|------|-------------|
| **Page CRO** | Landing page, homepage, pricing page optimization |
| **Signup Flow CRO** | Registration and sign-up optimization |
| **Onboarding CRO** | User activation and first-run experience |
| **Form CRO** | Form design and completion optimization |
| **Popup CRO** | Modal and popup optimization |
| **Paywall CRO** | Upgrade and paywall screen optimization |
| **Churn Prevention** | Cancel flow and dunning optimization |
| **A/B Testing** | Test setup, sample size, and analysis |
| **Analytics** | Event tracking and implementation |

## Quick Reference Commands

```
/cro page <url>         # Full page CRO audit
/cro signup <url>       # Signup flow analysis
/cro onboarding <url>   # Onboarding flow analysis
/cro form <url>         # Form optimization
/cro popup              # Popup strategy recommendations
/cro paywall            # Paywall optimization
/cro churn              # Churn prevention audit
/cro abtest             # A/B test setup guidance
/cro tracking           # Analytics implementation
```

## Usage

Before providing recommendations, I will:

1. **Check context**: Read `.claude/product-marketing-context.md` if it exists
2. **Identify page type**: Homepage, landing page, pricing, feature, etc.
3. **Understand goals**: Primary conversion goal and traffic sources
4. **Analyze systematically**: Using the CRO framework

## Output Format

Recommendations are structured as:
- **Quick Wins**: Easy changes with immediate impact
- **High-Impact Changes**: Bigger changes to prioritize
- **Test Ideas**: Hypotheses worth A/B testing
- **Copy Alternatives**: 2-3 alternatives for key elements
