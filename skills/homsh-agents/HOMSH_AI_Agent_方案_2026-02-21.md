## 一、总体定位

虹识技术是一家**技术密集型、小而精**的高科技公司。AI 智能体建设的核心原则：

> **不求大而全，求快而准——用智能体把30人的能力放大到300人。**

### 团队全景（30人）

| 部门 | 人数 | 核心工作 | 配备 Agent |
|------|------|---------|-----------|
| 算法研发 | 5人 | 虹膜识别算法、Phaselirs™ | IrisResearcher、CodeCopilot |
| 软件研发 | 5人 | 嵌入式、上位机、云平台 | EmbedCopilot、PlatformDev |
| 芯片研发 | 4人 | FPGA/ASIC 乾芯系列 | ChipDev、HardwareDoc |

## 三、各部门智能体设计

### 🧠 算法研发团队（5人）——技术核心

**方向**：虹膜识别算法 Phaselirs™、AI 算法升级（ViT、自监督学习）
**痛点**：前沿论文追踪靠人工、算法验证代码质量参差、技术文档滞后

#### Agent 1：算法研究助手 `IrisResearcher`
```
功能：
- 每日自动追踪 arXiv / IEEE / ACM 虹膜识别最新论文
- 提取关键技术点，与 Phaselirs™ 现有方案对比分析
- 生成"每周技术雷达"推送给算法团队
- 竞品算法动态监控（中科虹霸、IriTech 等）

工具栈：
- Brave Search + Tavily（论文与技术搜索）
- Claude Opus（深度分析）
- 企业微信/钉钉（自动推送）

数据节点：新加坡（公开论文数据）
```

#### Agent 2：算法代码助手 `CodeCopilot`
```
功能：
- 虹膜算法 Python/C++ 代码补全与审查
- 深度学习训练代码辅助（PyTorch/TensorFlow）
- 自动生成单元测试与性能基准测试
- 算法实验报告自动整理（准确率、EER、速度对比）
- Git 提交信息规范化

工具栈：
- Claude Code（主力）
- DeepSeek-Coder（本地备选，敏感算法代码）
- GitHub/GitLab

数据节点：武汉（核心算法代码严格本地）
```

---

工具栈：
- Claude Code（主力）
- DeepSeek-Coder（本地，敏感固件）

### 🔲 芯片研发团队（4人）——硬核竞争壁垒

**方向**：乾芯系列 FPGA/ASIC 虹膜识别智能芯片
**痛点**：HDL 代码审查耗时、规格文档维护难、IP 核选型复杂、时序分析报告繁琐

#### Agent 5：芯片开发助手 `ChipDev`
```
功能：
- Verilog/VHDL/SystemVerilog 代码补全与审查
- FPGA 时序约束文件（XDC/SDC）生成辅助
- IP 核选型建议（Xilinx/Intel FPGA IP 库检索）
- 功耗分析报告解读与优化建议
- ASIC 设计流程文档辅助（RTL→综合→布局布线）
- 芯片测试向量自动生成

工具栈：
- Claude（HDL 分析，理解 RTL 逻辑）
- DeepSeek（本地，敏感芯片设计）
- 本地知识库（历史芯片设计文档）

数据节点：武汉（芯片设计绝对不出境）
```

#### Agent 6：硬件文档助手 `HardwareDoc`
```
功能：
- 芯片数据手册（Datasheet）自动生成框架

- BOM 分析与供应商比价（立创商城、Digi-Key、Mouser）
- 器件替代型号智能推荐
- 硬件验收测试报告模板化输出

工具栈：
- Claude（文档生成）
- 本地知识库（历史规格书）
- 供应商 API

数据节点：武汉
```

## 五、实施路线图

### 第一阶段：地基（1-2个月）
**目标**：工具标准化 + 数据基础设施

- [ ] 新加坡节点部署 OpenClaw + Claude API
- [ ] 武汉节点部署 DeepSeek-V3（私有化）
- [ ] 建立企业知识库（产品文档、技术规格向量化）
- [ ] 研发团队 Claude Code 统一配置
- [ ] 制定数据分类规范（哪些数据走新加坡，哪些留武汉）

**可见成果**：研发团队人均 AI 编码效率提升 30%+

---

### 第二阶段：智能体上线（2-4个月）
**目标**：核心 Agent 部署，覆盖主要痛点

优先级排序（影响力 × 难度）：

| 优先级 | Agent | 影响力 | 难度 | 受益团队 |
|--------|-------|--------|------|---------|
| ⭐⭐⭐ | `IrisResearcher` | 高 | 低 | 算法研发 → **第一个上** |
| ⭐⭐⭐ | `CodeCopilot` | 高 | 低 | 算法研发 |
| ⭐⭐⭐ | `EmbedCopilot` | 高 | 低 | 嵌入式团队 |
| ⭐⭐⭐ | `PlatformDev` | 高 | 中 | 软件团队 |
| ⭐⭐ | `SalesBot` | 高 | 中 | 销售 |
| ⭐⭐ | `QABot` | 高 | 中 | 终端产品团队 |
| ⭐⭐ | `DocAgent` | 中 | 低 | 运营 |
| ⭐⭐ | `ProductDesigner` | 中 | 中 | 终端产品团队 |
| ⭐ | `HardwareDoc` | 高 | 高 | 芯片/FPGA（专业性强，后期上）|

**可见成果**：文档生成效率提升 5x，技术追踪自动化

---

### 第三阶段：协同进化（4-6个月）
**目标**：Agent 之间联动，形成闭环

- 研究 → 代码：IrisResearcher 发现新算法 → 自动生成代码框架
- 销售 → 运营：SalesBot 签单 → ProjectPilot 自动建项目
- 市场 → 融资：MarketRadar 数据 → 自动更新 BP 中的市场数据

**可见成果**：形成虹识技术专属的 AI 驱动研发运营闭环
