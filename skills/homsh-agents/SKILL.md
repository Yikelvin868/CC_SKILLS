# HOMSH AI Agent 体系 — 虹识技术 30人团队智能体方案 v3.0

## 概述

为武汉虹识技术有限公司（HOMSH）30人团队设计的 14个 AI 智能体，按部门划分，核心原则：**用智能体把30人的能力放大到300人**。

---

## 团队全景

| 部门 | 人数 | Agent |
|------|------|-------|
| 算法研发 | 5人 | IrisResearcher、CodeCopilot |
| 软件研发 | 5人 | EmbedCopilot、PlatformDev |
| 芯片研发 | 4人 | ChipDev、HardwareDoc |
| 终端研发 | 4人 | ProductDesigner、QABot |
| 销售 | 5人 | SalesBot、MarketRadar |
| 运营 | 3人 | DocAgent、ProjectPilot |
| 行政 | 2人 | AdminBot |
| 财务 | 2人 | FinanceBot |

---

## 数据安全边界

- **新加坡节点**：国际业务、对外客户沟通、非核心算法辅助
- **武汉节点**：核心算法、芯片设计、内部财务、专利文档（绝对不出境）

---

## 14个智能体详细规格

### 1. IrisResearcher — 算法研究助手
**部门**：算法研发 | **节点**：新加坡

**功能**：
- 每日自动追踪 arXiv / IEEE / ACM 虹膜识别最新论文
- 提取关键技术点，与 Phaselirs™ 现有方案对比分析
- 生成"每周技术雷达"推送给算法团队
- 竞品算法动态监控（中科虹霸、IriTech 等）

**工具栈**：Brave Search + Tavily、Claude Opus、企业微信推送

---

### 2. CodeCopilot — 算法代码助手
**部门**：算法研发 | **节点**：武汉（核心算法代码严格本地）

**功能**：
- 虹膜算法 Python/C++ 代码补全与审查
- 深度学习训练代码辅助（PyTorch/TensorFlow）
- 自动生成单元测试与性能基准测试
- 算法实验报告自动整理（准确率、EER、速度对比）
- Git 提交信息规范化

**工具栈**：Claude Code（主力）、DeepSeek-Coder（本地备选）

---

### 3. EmbedCopilot — 嵌入式开发助手
**部门**：软件研发 | **节点**：武汉

**功能**：
- 嵌入式 C/C++ 代码生成与审查（ARM Cortex-M/A）
- RTOS（FreeRTOS/RT-Thread）任务调度辅助
- 摄像头/LED/串口驱动模板生成
- 寄存器配置从数据手册自动提取
- 固件版本 Release Notes 自动生成

**工具栈**：Claude Code、DeepSeek-Coder（本地，敏感固件）

---

### 4. PlatformDev — 上位机 & 云平台助手
**部门**：软件研发 | **节点**：新加坡（云平台）/ 武汉（本地SDK）

**功能**：
- 上位机 GUI 代码生成（Qt/PyQt/Electron）
- REST API 接口文档自动生成（OpenAPI/Swagger）
- 云平台功能模块（设备管理、数据上报、身份认证）
- 自动化测试用例生成
- SDK 多版本兼容性检查

**工具栈**：Claude Code、GitHub Actions

---

### 5. ChipDev — 芯片开发助手
**部门**：芯片研发 | **节点**：武汉（芯片设计绝对不出境）

**功能**：
- Verilog/VHDL/SystemVerilog 代码补全与审查
- FPGA 时序约束文件（XDC/SDC）生成辅助
- IP 核选型建议（Xilinx/Intel FPGA IP 库检索）
- 功耗分析报告解读与优化建议
- ASIC 设计流程文档辅助（RTL→综合→布局布线）
- 芯片测试向量自动生成

**工具栈**：Claude（HDL 分析）、DeepSeek（本地）、本地知识库

---

### 6. HardwareDoc — 硬件文档助手
**部门**：芯片研发 | **节点**：武汉

**功能**：
- 芯片数据手册（Datasheet）自动生成框架
- PCB 设计评审 Checklist（EMC、SI、PI）
- BOM 分析与供应商比价（立创商城、Digi-Key、Mouser）
- 器件替代型号智能推荐
- 硬件验收测试报告模板化输出

**工具栈**：Claude、本地知识库、供应商 API

---

### 7. ProductDesigner — 产品设计助手
**部门**：终端研发 | **节点**：武汉

**功能**：
- 虹膜相机光学参数计算辅助（焦距、景深、照明功率）
- 产品规格书（PRD）自动生成
- 硬件设计评审 Checklist（EMC、散热、防护等级）
- 认证资料准备（CE、FCC、CCC）清单生成
- 竞品硬件规格对比分析

**工具栈**：Claude、本地知识库、供应商数据库

---

### 8. QABot — 测试与质量助手
**部门**：终端研发 | **节点**：武汉

**功能**：
- 测试方案自动生成（功能测试、环境测试、寿命测试）
- 测试报告模板化输出（GB/T 标准格式）
- 缺陷分析报告自动整理（按模块归类）
- 生产 SOP 文档生成与更新
- 供应商来料检验标准制定辅助

**工具栈**：Claude、Python（数据分析）、Excel/WPS

---

### 9. SalesBot — 销售方案助手
**部门**：销售 | **节点**：新加坡（国际客户）/ 武汉（国内客户）

**功能**：
- 根据客户行业（金融/海关/政府/军队）自动生成定制化产品方案
- 竞品分析（中科虹霸、聚虹智能等）
- 客户跟进提醒和 CRM 记录
- 英文/中文双语报价单生成

**工具栈**：Claude Sonnet（方案撰写）、GPT-4o（英文客户）、飞书/钉钉 CRM

---

### 10. MarketRadar — 展会市场情报助手
**部门**：销售 | **节点**：新加坡

**功能**：
- 监控安防展、生物识别行业展会信息
- 竞争对手动态追踪（融资、新产品、新合作）
- 行业政策法规变化预警（安防标准、数据安全法）
- 每周自动生成市场简报

**工具栈**：Tavily Search（深度搜索）、微信公众号监控

---

### 11. DocAgent — 文档助手
**部门**：运营 | **节点**：两地均可

**功能**：
- 技术白皮书、产品手册自动生成
- 标准化文档模板库（产品规格、测试报告、用户手册）
- 多格式输出：Word / PDF / PPT / 微信公众号排版

**工具栈**：Claude、nano-pdf / python-docx、Gamma

---

### 12. ProjectPilot — 项目协同助手
**部门**：运营 | **节点**：武汉

**功能**：
- 会议纪要自动生成（语音转文字→结构化）
- 项目进度看板自动更新
- 跨部门任务追踪和提醒

**工具栈**：Whisper（语音转文字）、Claude（结构化总结）、飞书/钉钉 API

---

### 13. AdminBot — 行政助手
**部门**：行政 | **节点**：武汉

**功能**：
- 差旅报销单自动整理
- 合同关键条款提取和风险提示
- 政府申报材料（高新认定、人才申报）模板生成

**工具栈**：Claude、OCR（票据识别）

---

### 14. FinanceBot — 财务助手
**部门**：财务 | **节点**：武汉（财务数据严格本地）

**功能**：
- 财务报表自动生成（月报/季报）
- 融资材料中的财务数据更新
- 成本分析（研发投入 vs 产出追踪）

**工具栈**：Claude（分析）、Python（数据处理）、Excel/WPS

## 调度架构

太極龍（OpenClaw）作为调度中枢，统一管理所有 Agent：
- 定时任务：IrisResearcher（每日）、MarketRadar（每周）
- 按需调用：研发类、业务类、支撑类 Agent
- 统一权限管理、使用量监控、多通道接入（企业微信/钉钉/网页）
