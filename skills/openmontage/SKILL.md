# OpenMontage — AI Video Production Studio

## Overview

World's first open-source agentic video production system. 11 pipelines, 49 tools, 400+ agent skills. Turn Claude Code into a full video production studio.

## When to Use

- Professional video production (cinematic, explainer, talking-head, animation)
- Video editing with multiple shots, transitions, effects
- Screen demo recording and editing
- Podcast repurposing to video
- Avatar/spokesperson videos
- Video localization and dubbing
- Any task that requires Remotion-based video rendering

## Installation

Installed at: ~/Projects/OpenMontage/

## Pipelines Available

| Pipeline | Description |
|----------|-------------|
| cinematic | Full cinematic production |
| animated-explainer | Animated explainer videos |
| talking-head | Talking head with captions |
| avatar-spokesperson | AI avatar spokesperson |
| clip-factory | Short-form clip generation |
| screen-demo | Screen recording demos |
| podcast-repurpose | Podcast → video conversion |
| localization-dub | Multi-language dubbing |
| hybrid | Mixed format production |
| animation | Pure animation |

## How to Use

Load the full agent guide from the OpenMontage project:
```
Read ~/Projects/OpenMontage/AGENT_GUIDE.md
Read ~/Projects/OpenMontage/AGENTS.md
```

Then follow the routing rules in AGENT_GUIDE.md.

## Configuration

- Config: ~/Projects/OpenMontage/config.yaml
- Pipeline defs: ~/Projects/OpenMontage/pipeline_defs/
- Output: 1920x1080, 30fps, MP4 (H.264)
