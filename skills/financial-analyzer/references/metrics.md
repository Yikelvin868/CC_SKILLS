## 盈利能力指标

### 毛利率 (Gross Margin)
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```
- **含义**：反映产品或服务的基本盈利能力
- **优秀**：>40%（软件/医药）、>30%（消费品）、>20%（制造业）
- **警示**：持续下降可能表明竞争加剧或成本上升

### 净利率 (Net Profit Margin)
```
净利率 = 净利润 / 营业收入 × 100%
```
- **含义**：最终盈利能力，考虑所有成本和费用
- **优秀**：>15%（高端服务）、>10%（消费品）、>5%（零售）
- **注意**：需排除非经常性损益影响

### ROE - 净资产收益率 (Return on Equity)
```
ROE = 净利润 / 平均股东权益 × 100%
平均股东权益 = (期初权益 + 期末权益) / 2
```
- **含义**：股东投入资本的回报率，巴菲特最看重的指标
- **优秀**：>20% 为优秀，>15% 为良好
- **杜邦分解**：ROE = 净利率 × 资产周转率 × 权益乘数

### ROA - 总资产收益率 (Return on Assets)
```
ROA = 净利润 / 平均总资产 × 100%
```
- **含义**：资产利用效率
- **优秀**：>10%（轻资产）、>5%（重资产）
- **与ROE对比**：差距大说明杠杆高

### ROIC - 投入资本回报率 (Return on Invested Capital)
```
ROIC = NOPAT / 投入资本 × 100%
NOPAT = 营业利润 × (1 - 税率)
投入资本 = 股东权益 + 有息负债 - 超额现金
```
- **含义**：衡量公司核心业务的资本效率，排除资本结构影响
- **优秀**：ROIC > WACC 表示创造价值
- **最佳指标**：跨公司比较的最佳盈利指标

## 成长性指标

### 同比增长率 (YoY Growth)
```
YoY = (本期值 - 去年同期值) / 去年同期值 × 100%
```
- **应用**：营收YoY、净利润YoY、毛利YoY
- **注意**：需考虑基数效应和季节性

### 复合年增长率 (CAGR)
```
CAGR = (期末值/期初值)^(1/n) - 1
n = 年数
```
- **应用**：评估长期增长趋势，平滑短期波动
- **优秀**：营收 CAGR >15%、净利润 CAGR >20%

### 环比增长率 (QoQ Growth)
```
QoQ = (本季度值 - 上季度值) / 上季度值 × 100%
```
- **应用**：观察短期趋势变化
- **注意**：季节性行业需谨慎解读

## 偿债能力指标

### 资产负债率 (Debt to Assets)
```
资产负债率 = 总负债 / 总资产 × 100%
```
- **安全区间**：30%-60%（行业差异大）
- **警示**：>70% 需关注偿债风险

### 流动比率 (Current Ratio)
```
流动比率 = 流动资产 / 流动负债
```
- **健康值**：1.5-2.0
- **含义**：短期偿债能力

### 速动比率 (Quick Ratio)
```
速动比率 = (流动资产 - 存货) / 流动负债
```
- **健康值**：>1.0
- **含义**：剔除存货后的即时偿债能力

### 利息保障倍数 (Interest Coverage)
```
利息保障倍数 = EBIT / 利息费用
EBIT = 营业利润 + 利息费用
```
- **安全值**：>3 倍
- **警示**：<1.5 倍存在偿债风险

### 净负债率 (Net Debt to Equity)
```
净负债率 = (有息负债 - 现金) / 股东权益 × 100%
```
- **含义**：考虑现金储备后的真实杠杆水平
- **负值**：净现金状态，财务安全

## 运营效率指标

### 应收账款周转天数 (DSO)
```
DSO = 365 / (营业收入 / 平均应收账款)
平均应收账款 = (期初 + 期末) / 2
```
- **含义**：收款效率
- **优秀**：<45天（B2C）、<90天（B2B）
- **警示**：持续上升可能表明坏账风险

### 存货周转天数 (DIO)
```
DIO = 365 / (营业成本 / 平均存货)
```
- **含义**：存货管理效率
- **优秀**：零售 <60天、制造 <90天
- **警示**：上升可能表明滞销

### 应付账款周转天数 (DPO)
```
DPO = 365 / (营业成本 / 平均应付账款)
```
- **含义**：付款周期，议价能力体现
- **健康**：与行业惯例匹配

### 现金周期 (CCC)
```
CCC = DSO + DIO - DPO
```
- **含义**：从付款到收款的时间跨度
- **优秀**：负数表示无运营资金需求（如京东）
- **警示**：CCC 过长会占用大量资金

### 总资产周转率 (Asset Turnover)
```
总资产周转率 = 营业收入 / 平均总资产
```
- **含义**：资产利用效率
- **对比**：轻资产 >1.5，重资产 <0.5

## 现金流指标

### 经营现金流/净利润
```
比率 = 经营活动现金流净额 / 净利润
```
- **优秀**：>1.0 表示盈利质量高
- **警示**：<0.8 或持续<1 需警惕应收账款或存货问题

### 自由现金流 (FCF)
```
FCF = 经营活动现金流净额 - 资本支出
资本支出 = 购建固定资产、无形资产支付的现金
```
- **含义**：可自由支配的现金
- **重要性**：巴菲特称之为"股东盈余"

### 自由现金流收益率 (FCF Yield)
```
FCF Yield = 自由现金流 / 市值 × 100%
```
- **优秀**：>5% 表示估值有吸引力
- **应用**：现金流估值的重要指标

### 现金流覆盖率
```
现金流覆盖率 = 经营现金流 / 流动负债
```
- **安全值**：>0.4
- **含义**：经营产生现金偿还短期债务的能力

| 消费品 | 40-60% | 10-15% | 15-25% | 40-60% |

| 制造业 | 15-30% | 5-10% | 10-15% | 40-60% |
| 银行 | N/A | 25-35% | 10-15% | 90%+ |
| 房地产 | 25-35% | 8-15% | 10-20% | 70-80% |

**注意**：以上为一般参考值，需结合具体公司和市场情况分析。
