## 相对估值法

### P/E - 市盈率 (Price to Earnings)
```
P/E = 股价 / 每股收益 = 市值 / 净利润
```

**类型**：
- **TTM P/E**：过去12个月净利润（滚动）
- **Forward P/E**：预测未来12个月净利润
- **静态 P/E**：最近财年净利润

**估值区间参考**：
| 类型 | 低估 | 合理 | 高估 |
|------|------|------|------|
| 成长股 | <20 | 20-40 | >40 |
| 价值股 | <10 | 10-15 | >15 |
| 周期股 | 高P/E买 | - | 低P/E卖 |

**PEG 估值**：
```
PEG = P/E / 盈利增速(%)
```
- PEG < 1：可能被低估
- PEG = 1：合理估值
- PEG > 1：可能被高估

### P/B - 市净率 (Price to Book)
```
P/B = 股价 / 每股净资产 = 市值 / 股东权益
```

**适用**：银行、保险、重资产行业

**估值区间**：
- 银行：0.5-1.5
- 保险：1.0-2.0
- 制造业：1.0-3.0

**ROE-PB 关系**：
```
合理 P/B ≈ ROE / 资本成本
```
ROE 高的公司应享有更高 P/B

### P/S - 市销率 (Price to Sales)
```
P/S = 市值 / 营业收入
```

**适用**：
- 高增长但尚未盈利的公司
- SaaS、互联网公司
- 周期底部的公司

**参考**：
- 成熟公司：1-3
- 高增长公司：5-15
- SaaS：10-20+（取决于增速）

### EV/EBITDA
```
EV = 市值 + 有息负债 - 现金
EBITDA = 营业利润 + 折旧 + 摊销
EV/EBITDA = 企业价值 / EBITDA
```

**优势**：
- 排除资本结构差异
- 排除折旧政策差异
- 适合跨国、跨行业对比

**参考区间**：
| 行业 | 低估 | 合理 | 高估 |
|------|------|------|------|
| 科技 | <10 | 10-20 | >20 |
| 消费 | <8 | 8-12 | >12 |
| 工业 | <6 | 6-10 | >10 |

### EV/Sales
```
EV/Sales = 企业价值 / 营业收入
```
- 类似 P/S，但考虑负债
- 适合高负债公司对比

## DCF 估值模型

### 基本公式
```
企业价值 = Σ(FCF_t / (1+WACC)^t) + 终值/(1+WACC)^n

FCF = 自由现金流
WACC = 加权平均资本成本
t = 预测年份
n = 预测期末年份
```

### 第一步：预测自由现金流

**自由现金流计算**：
```
FCF = EBIT × (1-税率) + 折旧摊销 - 资本支出 - 营运资本增加
```

**预测方法**：
1. 营收增长假设（基于历史、行业、管理层指引）
2. 利润率假设（趋势、竞争格局）
3. 资本支出假设（维持性 + 扩张性）
4. 营运资本假设（与营收联动）

**典型预测期**：5-10年

### 第二步：计算 WACC
```
WACC = E/(D+E) × Ke + D/(D+E) × Kd × (1-T)

E = 股权价值（市值）
D = 债务价值
Ke = 股权成本
Kd = 债务成本
T = 税率
```

**股权成本 (CAPM)**：
```
Ke = Rf + β × (Rm - Rf)

Rf = 无风险利率（10年国债收益率）
β = 贝塔系数
Rm - Rf = 市场风险溢价（通常 5-7%）
```

**典型 WACC 范围**：8-12%

### 第三步：计算终值

**永续增长模型**：
```
终值 = FCF_n × (1+g) / (WACC - g)
g = 永续增长率（通常 2-3%，不超过 GDP 增速）
```

**退出倍数法**：
```
终值 = EBITDA_n × 目标 EV/EBITDA
```

### 第四步：计算现值
```
企业价值 = Σ(FCF_t 现值) + 终值现值
股权价值 = 企业价值 - 净负债
每股价值 = 股权价值 / 总股数
```

### 敏感性分析

建议对以下参数做敏感性分析：
- WACC：±1%
- 永续增长率：±0.5%
- 营收增速：±2%
- 利润率：±1%

**敏感性矩阵示例**：
| WACC \ g | 2.0% | 2.5% | 3.0% |
|----------|------|------|------|
| 9% | $XX | $XX | $XX |
| 10% | $XX | $XX | $XX |
| 11% | $XX | $XX | $XX |

## 估值判断框架

### 综合判断流程

1. **历史对比**
   - 当前估值 vs 历史均值/中位数
   - 当前估值 vs 历史区间（25%-75%分位）

2. **同业对比**
   - 与直接竞争对手对比
   - 与行业平均水平对比
   - 考虑增速差异调整

3. **绝对估值**
   - DCF 模型计算内在价值
   - 与当前股价对比

### 估值评级标准

| 评级 | 条件 |
|------|------|
| **明显低估** | 当前价格 < 内在价值 × 70% |
| **低估** | 当前价格 < 内在价值 × 85% |
| **合理** | 内在价值 × 85% ≤ 当前价格 ≤ 内在价值 × 115% |
| **高估** | 当前价格 > 内在价值 × 115% |
| **明显高估** | 当前价格 > 内在价值 × 130% |

### 安全边际考虑

根据业务确定性设置不同安全边际：
- **高确定性**（消费龙头、公用事业）：15-20%
- **中确定性**（制造业龙头）：25-30%
- **低确定性**（周期股、科技股）：35-50%

### 常见估值陷阱

1. **价值陷阱**：低估值但基本面恶化
2. **增长陷阱**：高估值且增速放缓
3. **周期陷阱**：周期顶部低 P/E 并非低估
4. **质量陷阱**：忽视盈利质量问题
