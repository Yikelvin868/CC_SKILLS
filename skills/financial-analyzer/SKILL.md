# 财务分析师 Skill

## 核心能力

1. **财报解析**：提取损益表、资产负债表、现金流量表关键数据
2. **财务指标计算**：盈利能力、成长性、偿债能力、运营效率
3. **估值分析**：相对估值（P/E、P/B、EV/EBITDA）、DCF 模型
4. **同行对比**：行业 benchmark 分析
5. **报告生成**：投资备忘录、分析报告、演示文稿

## 分析流程

### 第一步：数据收集

- 识别财报类型（年报/季报/10-K/10-Q）和报告期
- 提取三大财务报表核心数据
- 整理历史财务数据（3-5年）
- 如用户提供 Excel/CSV 文件，使用 `scripts/extract_financials.py` 提取数据

### 第二步：财务分析

使用 `scripts/calculate_metrics.py` 计算以下指标：

**盈利能力**
- 毛利率 = 毛利润 / 营收
- 净利率 = 净利润 / 营收
- ROE = 净利润 / 股东权益
- ROA = 净利润 / 总资产
- ROIC = NOPAT / 投入资本

**成长性**
- 营收 YoY = (本期营收 - 上期营收) / 上期营收
- 净利润 YoY = (本期净利润 - 上期净利润) / 上期净利润
- 3年 CAGR = (期末值/期初值)^(1/3) - 1

**偿债能力**
- 资产负债率 = 总负债 / 总资产
- 流动比率 = 流动资产 / 流动负债
- 速动比率 = (流动资产 - 存货) / 流动负债
- 利息保障倍数 = EBIT / 利息费用

**运营效率**
- 应收周转天数 = 365 / (营收/平均应收账款)
- 存货周转天数 = 365 / (营业成本/平均存货)
- 应付周转天数 = 365 / (营业成本/平均应付账款)
- 现金周期 = 应收周转天数 + 存货周转天数 - 应付周转天数

**现金流质量**
- 经营现金流/净利润（>1 表示盈利质量高）
- 自由现金流 = 经营现金流 - 资本支出

详细指标公式和行业基准参见 [references/metrics.md](references/metrics.md)

### 第三步：估值分析

**相对估值**
- P/E = 市值 / 净利润
- P/B = 市值 / 净资产
- P/S = 市值 / 营收
- EV/EBITDA = 企业价值 / EBITDA

**估值对比**
- 对比历史估值区间（3-5年）
- 同行业公司估值对比
- 判断当前估值位置

**DCF 估值**（如需要）
使用 `scripts/dcf_model.py` 构建简化 DCF 模型：
- 预测未来 5 年自由现金流
- 确定折现率（WACC）
- 计算终值和现值

估值方法详解参见 [references/valuation.md](references/valuation.md)

### 第四步：投资结论

输出结构化投资结论：
- **核心投资逻辑**：3-5 个关键投资理由
- **主要风险点**：需要关注的风险因素
- **估值判断**：低估 / 合理 / 高估
- **建议行动**：买入 / 持有 / 卖出 / 观望

## 输出格式

根据用户需求选择输出格式：

| 格式 | 用途 | 调用方式 |
|------|------|----------|
| Excel | 详细财务数据表、估值模型 | 调用 xlsx skill |
| Word | 结构化分析报告 | 调用 docx skill |
| PowerPoint | 投资委员会演示 | 调用 pptx skill |
| Markdown | 快速分析摘要 | 直接输出 |
