# 虹识技术 VI 规范参考

## 品牌色彩

| 用途 | CMYK | RGB (近似) | HEX |
|------|------|------------|-----|
| 品牌红（主色） | C5 M100 Y100 K0 | RGB(230, 0, 18) | #E60012 |
| 深黑/深棕黑 | C30 M30 Y30 K100 | RGB(35, 31, 32) | #231F20 |
| 中性灰（辅助） | - | RGB(128, 128, 128) | #808080 |
| 白色 | - | RGB(255, 255, 255) | #FFFFFF |

## 字体系统

### 中文字体：思源黑体 (Source Han Sans)
- **Heavy**: 封面标题、大标题
- **Medium**: 小标题
- **Light/Regular**: 正文

### 英文字体
无衬线、几何感、风格克制的字体（如 Source Sans Pro）

### 字号层级（PPT）
| 层级 | 字号 |
|------|------|
| 封面标题 | 40-48 pt |
| 一级标题 | 28-32 pt |
| 二级标题 | 20-24 pt |
| 正文 | 16-18 pt |
| 注释 | ≤14 pt |

## 背景规范

### 允许的背景组合
- 纯红底 + 白色文字
- 纯黑底 + 白色/红色文字
- 浅灰底 + 深灰/黑色文字
- 白色底 + 黑色/红色文字

### 禁止使用
- 渐变色背景
- 蓝色/科幻蓝调
- 多色混合背景

## LOGO 使用规范

### 位置规则
| 页面类型 | LOGO位置 |
|----------|----------|
| 封面页 | 居中 或 上中 |
| 内容页 | 左下 / 右下 |
| 封底页 | 居中 |

### 安全空间
LOGO周围保留不少于 3a 的安全空间（a为VI模数单位）

### 禁止操作
- 不得描边、投影、渐变
- 不得与复杂图片叠加
- 只能使用标准LOGO文件

## 辅助图形

### 允许使用
- 虹膜纹理抽象化
- 横向红色扫描线（品牌特色元素）
- 环形裁切框
- 放射状、环形图形

### 禁止使用
- 圆角卡片风格
- 拟物化图标
- 花哨装饰线条

## 页面类型指引

### 封面页
- 纯红或纯黑背景
- LOGO + 标题（≤2行）
- 可添加红色扫描线装饰

### 内容页
- 灰色或白色背景
- 黑色文字为主
- 红色仅用于强调关键点

### 技术架构页
- 灰/白背景
- 清晰的层次结构
- 红色标注关键模块

### 结束页
- LOGO居中
- 可配品牌Slogan
- 保持干净、充分留白
- 公司网址: www.homsh.cn

## 设计禁区

严格禁止以下风格：
- ❌ 蓝色科技风
- ❌ 卡通/插画风格
- ❌ 花哨图标
- ❌ 多字体混用
- ❌ 过度动画效果
- ❌ 渐变背景

## 核心设计理念

三大特质：**克制、力量感、秩序感**

- 克制：大面积留白、纯色运用、拒绝视觉噪音
- 力量感：品牌红与深黑的高对比组合
- 秩序感：严格网格、精确比例、一致对齐

品牌认知：**生物识别·可信身份**
