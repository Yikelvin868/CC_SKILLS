# HOMSH Social Media Publisher

When the user wants to publish HOMSH product content to social media platforms. Trigger when user mentions "publish to social", "post to LinkedIn", "post to X", "post to YouTube", "发社媒", "发布到", "社交媒体发布", or needs to distribute HOMSH marketing content across platforms.

## Overview

Publishes HOMSH product videos and ad copy to multiple social media platforms (X, LinkedIn, YouTube, WeChat). Handles platform-specific formatting, video requirements, and scheduling.

## Prerequisites

- Chrome browser with CDP enabled (web-access skill)
- User logged into target platforms in Chrome
- Video file ready (processed by homsh-brand-overlay skill)
- Ad copy ready (generated by homsh-ad-copy skill)

## Supported Platforms

| Platform | Video Format | Max Duration | Max File Size | Aspect Ratio |
|----------|-------------|-------------|---------------|--------------|
| X / Twitter | MP4 | 2:20 | 512MB | Any (16:9, 9:16, 1:1) |
| LinkedIn | MP4 | 10:00 | 5GB | 16:9, 1:1, 9:16 |
| YouTube | MP4 | 12:00:00 | 256GB | 16:9 preferred |
| WeChat | MP4 | 30:00 | 1GB | Any |

## Workflow

### Step 1: Pre-flight Check

1. Verify video file exists and is valid MP4
2. Check file size against platform limits
3. Confirm ad copy is ready for target platform
4. Load web-access skill for browser automation

### Step 2: Platform-Specific Publishing

#### X / Twitter

```
1. Navigate to https://x.com/compose/post
2. Verify login state
3. Upload video via file input
4. Input text copy (280 char limit)
5. Click "Post" button
6. Confirm success
```

**CDP selectors (may change):**
- Compose URL: `https://x.com/compose/post`
- File input: `input[type=file][accept*=video]`
- Text area: `[data-testid="tweetTextarea_0"]`
- Post button: `[data-testid="tweetButton"]`

#### LinkedIn

```
1. Navigate to https://www.linkedin.com/feed/
2. Verify login state
3. Click "Start a post" button
4. Click video/media upload icon
5. Upload video file
6. Input text copy
7. Click "Post" button
8. Confirm success
```

#### YouTube

```
1. Navigate to https://studio.youtube.com/
2. Click "Create" > "Upload videos"
3. Upload video file
4. Fill in title, description, tags
5. Set visibility (Public/Unlisted/Scheduled)
6. Click "Publish" or "Schedule"
```

#### WeChat (via baoyu-post-to-wechat skill)

Delegate to existing `baoyu-post-to-wechat` skill for WeChat Official Account publishing.

### Step 3: Post-publish Verification

- Screenshot the published post
- Verify video is playing/processing
- Save post URL for tracking
- Report success to user

## Scheduling

For delayed publishing (e.g., "post to YouTube next Thursday"):

1. Save task to memory with absolute date
2. On target date, remind user or auto-publish if authorized
3. LinkedIn and YouTube both support native scheduling

## Multi-Platform Batch Publishing

When publishing to multiple platforms simultaneously:

```
Launch parallel sub-agents:
- Agent 1: Publish to X (via CDP)
- Agent 2: Publish to LinkedIn (via CDP)
- Agent 3: Publish to YouTube (via CDP)
Each agent loads web-access skill independently.
```

## Error Handling

| Error | Solution |
|-------|----------|
| Not logged in | Ask user to login in Chrome, then retry |
| Video upload fails | Check file size, try re-upload |
| Post blocked by platform | Review content for policy violations |
| Rate limited | Wait and retry, or schedule for later |
