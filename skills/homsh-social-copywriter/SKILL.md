# HOMSH Social Copywriter — 虹识社媒文案生成技能

## Overview

为 HOMSH Technology 的产品视频和市场活动生成专业社媒文案，覆盖 LinkedIn、X/Twitter、YouTube 描述。针对 B2B 安防/生物识别行业的海外获客场景优化。

## When to Use This Skill

- 产品视频制作完成后，需要配套社媒发布文案
- 新产品发布、展会参展、案例分享等市场活动
- 需要多平台（LinkedIn / X / YouTube）一次性生成文案

## Tone & Voice

- **专业但不刻板** — 安防行业 B2B 决策者视角
- **问题导向** — 先讲痛点，再给方案
- **不用 emoji** — 保持专业严肃
- **数据驱动** — 引用具体技术参数（"266 个虹膜特征点 vs 指纹 40 个"）
- **不用 AI 腔调** — 避免 "revolutionize"、"cutting-edge"、"game-changing" 等空洞词汇

## Platform Templates

### LinkedIn（B2B 主战场）

**结构：**
```
第1段：行业痛点 / 反直觉观点（hook）
第2段：空行
第3段："我们做了不同的事"（转折）
第4段：空行
第5段：产品核心价值（3-5 个要点，用 -- 开头）
第6段：空行
第7段：CTA + 视频链接
第8段：空行
第9段：联系方式
第10段：空行
第11段：hashtags
```

**字数：** 1200-1800 字符（LinkedIn 最佳长度）

**关键词策略：**
- 首行包含核心关键词（biometric access control）
- 结尾包含品牌名和联系方式
- hashtags 10-15 个，混合行业词 + 品牌词

**示例 Hook 开头：**
```
Most access control systems still rely on a single biometric factor.
That is a known vulnerability in high-security environments where 
tailgating, credential sharing, and spoofing are real threats.

We built something different.
```

### X / Twitter（品牌曝光 + 引流）

**单条版（< 280 字符）：**
```
[核心卖点一句话] + [产品名] + [关键差异] + [CTA 链接]
#tag1 #tag2
```

**Thread 版（3-4 条）：**
```
Tweet 1: Hook — 反直觉事实或安全漏洞
Tweet 2: Product — 产品如何解决问题
Tweet 3: Proof — 技术数据 / 对比数据
Tweet 4: CTA — 视频链接 + 联系方式 + hashtags
```

**Thread 发布规则：**
- Tweet 1 发布后，回复自己形成 thread
- 每条 tweet 独立可读
- 最后一条带链接和 hashtags

## Common Pitfalls

- 不要用 "cutting-edge"、"revolutionary" 等空洞词 → 用具体数据
- 不要用 emoji → B2B 安防行业保持严肃
- LinkedIn 不要超过 3000 字符 → 最佳 1200-1800
- X 单条不要超过 280 字符 → 链接算 23 字符（t.co 缩短）
- 不要忘记 CTA → 每条都要有明确的下一步动作
- 联系方式用 opticsiris.com（不是 homshtech.com）
