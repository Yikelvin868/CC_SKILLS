# HOMSH Video Ad Prompt Templates

## Scene Templates

### 1. Office Lobby (Default)
```
镜头缓缓推进，展示这台{product}智能门禁设备的精致外观，屏幕微微亮起，背景为现代化办公大厅，光线柔和，科技感强烈
```

### 2. Building Entrance
```
高端写字楼玻璃门旁，{product}门禁终端固定在墙面上，一位商务人士走近，屏幕自动亮起识别界面，门缓缓打开，科技感十足
```

### 3. Data Center
```
数据中心入口，蓝色冷光环境中，{product}门禁设备屏幕发出柔和光芒，展示虹膜识别过程，安全门打开，专业感强烈
```

### 4. Smart Campus
```
现代化校园走廊，{product}智能终端安装在教室门旁，学生走近自动识别通过，阳光透过窗户洒落，温暖而智能
```

### 5. Product Showcase (Pure)
```
纯黑背景中，{product}设备缓缓旋转展示，屏幕依次亮起各功能界面，光线勾勒出设备精致线条，高端产品广告感
```

### 6. Close-up Detail
```
特写镜头，{product}设备屏幕上功能图标依次亮起：生物识别、密码开锁、刷卡，镜头缓缓拉远展示设备全貌
```

## Duration Recommendations

| Duration | Use Case | Credits |
